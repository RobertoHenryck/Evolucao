Primeiro Projeto

O projeto "Engine Tuner" foi desenvolvido com o objetivo de destacar as referências brasileiras na área de tuning e otimização de motores, englobando marcas, profissionais renomados e a relevância desse segmento. Utilizando HTML e CSS para estruturação e estilização, e PHP básico para funcionalidades dinâmicas, o site oferece uma experiência visual e informativa.
A página inicial apresenta uma introdução ao universo do engine tuning, abordando sua importância para performance, economia e personalização de veículos. Há uma seção dedicada às marcas nacionais que se destacam no mercado, mostrando sua inovação e qualidade. Além disso, um espaço exclusivo foi criado para homenagear os principais profissionais brasileiros, destacando suas contribuições e trajetórias.
Com um layout simples e funcional, o projeto buscou informar e valorizar a profissão de engine tuner, que combina paixão por motores e expertise técnica. Assim, o site reforça o papel desse profissional no crescimento do setor automotivo no Brasil.


Segundo Projeto

O projeto PodRecomenda foi desenvolvido utilizando PHP avançado, HTML e CSS, com o objetivo de criar uma plataforma interativa para amantes de podcasts. Ele oferece funcionalidades como apresentação de episódios e um sistema de cadastro e login para usuários.
A interface exibe podcasts recomendados, destacando detalhes como título, descrição e autores. Os usuários podem se registrar, fazer login e acessar recursos exclusivos, como criar playlists ou salvar episódios favoritos.
No backend, o PHP gerencia autenticação, validação de dados e integração com o banco de dados MySQL, garantindo segurança e eficiência. Senhas são armazenadas com criptografia, e sessões foram implementadas para proteger os acessos.
O design combina usabilidade e estilo, oferecendo uma experiência agradável e intuitiva. O PodRecomenda busca conectar pessoas a conteúdos de qualidade, tornando a descoberta e organização de podcasts uma tarefa simples e prazerosa.
